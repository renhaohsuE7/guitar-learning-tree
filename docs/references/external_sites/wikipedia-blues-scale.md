## 摘要

### Blues Scale 定義
- 最常見形式：Minor Pentatonic（五聲小調）+ ♭5（降五度）= 6音音階（Hexatonic）
- 因此教學順序：五聲音階 → 藍調音階 是合理的 ✅

### 樹狀圖音階順序驗證
目前順序：五聲音階 → 大調音階 → 小調音階 → 藍調音階 → Modes

| 順序 | 說明 | 合理性 |
|------|------|--------|
| 五聲音階優先 | 最常見入門音階，即便 blues 也依賴它 | ✅ 正確 |
| 大調/小調在藍調前 | 提供理論基礎，再學 blues 色彩音 | ✅ 合理 |
| Modes 最後 | 進階概念，需要大調基礎 | ✅ 正確 |

### 結論
音階學習順序正確，五聲音階優先完全符合主流教學邏輯。
