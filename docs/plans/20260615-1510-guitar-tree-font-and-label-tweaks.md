# Plan: 樹狀圖字體放大 + 移除圓圈數字

## 需求討論摘要
使用者直接指定兩個 UI 微調：
1. 所有節點的字體大小調大
2. 主幹節點名稱的圓圈數字（①②③④⑤⑥）改為一般數字（1. 2. 3. 4. 5. 6.）

## 目標
讓樹狀圖文字更易讀，標籤風格更簡潔。

## 設計 Spec

### 字體大小調整
| 節點類型 | 目前 | 調整後 |
|----------|------|--------|
| 根節點   | 15px | 18px   |
| 一般節點 | 12px | 14px   |
| 葉節點   | 11px | 13px   |

### 標籤替換
| 原文 | 替換為 |
|------|--------|
| ① 基礎入門 | 1. 基礎入門 |
| ② 基本技巧 | 2. 基本技巧 |
| ③ 音樂理論 | 3. 音樂理論 |
| ④ 音樂風格 | 4. 音樂風格 |
| ⑤ 設備知識 | 5. 設備知識 |
| ⑥ 實戰進階 | 6. 實戰進階 |

## 技術決策
- 字體大小改 CSS（`.node text`、`.node.n-root text`、`.node.n-leaf text`）
- 標籤替換直接改 JS data 物件中的 `name` 欄位

## 實作步驟
1. 修改 CSS 字體大小
2. 替換 data 物件中六個主幹的 name 字串
